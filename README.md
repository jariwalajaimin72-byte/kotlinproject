# Adorable Knots 🧶

Adorable Knots is a mobile application developed using Kotlin in Android Studio.  
This app is designed for crochet lovers to explore beautiful knot patterns, ideas, and creative designs.

# 📱 Features
- Browse different crochet knot designs
- Clean and user-friendly interface
- Easy navigation between patterns
- Attractive UI for better user experience

# 🛠️ Technologies Used
- Kotlin
- Android Studio
- XML (UI Design)
